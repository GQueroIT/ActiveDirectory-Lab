# Active Directory Lab Overview

## Objective
This lab documents my hands-on buildout of a small Active Directory environment in VirtualBox. The goal is to create a working Windows Server domain controller, validate core Active Directory services, organize the domain with administrative and workstation-focused OUs, and prepare a Windows 10 client for domain integration.

## Current Progress
Completed work so far includes:
- Repository and screenshot structure setup
- Git and GitHub workflow setup
- VirtualBox preparation
- Windows Server VM creation
- Network configuration and static IP setup
- Active Directory Domain Services installation
- Domain promotion to relentixtest.local
- Post-promotion validation
- Domain admin account creation
- OU structure creation
- Final Active Directory validation
- Windows 10 client installation

## Lab Naming
- Domain: relentixtest.local
- Domain Controller: DC01
- Windows Client: CLIENT01

## Repository Layout
- configs/ = configuration references
- documents/ = step-by-step documentation
- logs/ = validation notes
- screenshots/ = visual proof
- scripts/ = automation
